## お金を稼ぐというこは

最近広告で副業とかこのツールで月〇万稼ぐ方法とかをよく目にする。
なぜ自分で知識とノウハウを独り占めしないのかと思っていたが本質がそこにありそうだ。
こういった知識を使うより、知識やツールの使い方を販売するほうが儲かることに気づいた。高額のスクールを開設してみたり、企業の戦略などを代わりに考えるコンサルティングが異常に稼いでるように感じるのはこのせいだろう。


## どのような考え方が大切か

- マネタイズのゴールを作ること。
例えば月100万円作ろうとしたら5万円の商品なら20件、10万円の商品なら月10件
つまり、「顧客が何人いるか」が問題ではなく、「適切なオファー」を「適切な人」が何人見ているかだけ。
- 「 お金が支払われる痛み」を選ぶ。
自身のオーディエンスが抱えている「高くて、痛くて、放置できない問題」は何か。
例えば、「コンテンツの作り方を教える」これは問題ではなく、でも、「コンテンツからお金を生めていない」これは立派な問題
人はモチベーションにお金を払はない。人がお金を払うのは次の三つだけ
 1. お金
 2. 時間
 3. 確実性（安心/再現性）
だからオファーはこの3つのどれかに紐づける必要がある。

これらが**売れる** 商品の作り方。
無料コンテンツの役割は「**認知**」
優良オファーの役割は「**仕組み化**」

人が欲しいのは「**結果が出る保証**」
